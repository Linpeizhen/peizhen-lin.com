As a UI Designer

我是一位 UI 設計師，熱愛 UI 設計這個領域，也非常重視 UX 設計，執行設計時以「美觀易讀的介面」、「友善的使用者體驗」及「有效地解決根本問題」為目標，致力於美學及易用性的結合。

我參與過的專案橫跨服務各種對象及裝置的應用，擁有跨平台、跨領域設計的能力。例如跨平台的電商網站、App、系統後台、行銷活動頁和 EDM 等，其中 KKday 的 App 在 2018 年獲頒 Google 最具潛力應用程式。

過去與其他團隊成員合作的過程中，我樂於與不同職能的角色合作、參與討論，也能接納不同的想法和各方的評論。我也遇到許多優秀的 UI 夥伴，定期分享靈感、舉辦讀書會和工作坊，互相學習成長。
